What's this for?

This is a NBA game simulator. We assume the NBA game scores follow a poisson distribution
with some other parameters.

[reference]


How to use?

1.	Run the program:

(a)	iPython IDE with necessary packages under Windows:
	Open 'main.py' file
	Press F5 to run

(b)	Linux shell with python interpreter:
	Using cd ...to enter into the VaR/ES Calculation System directory
	Type 'python main.py' in the shell

3.	Go to 'Outputs' folder and check latest generated folders
	'VaR & ES.csv' and 'VaR Backtesting.csv' are final results.

The detailed description and all test results are saved in 'Documents' Folder.
